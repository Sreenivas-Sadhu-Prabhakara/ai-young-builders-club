## 💻 A taste of real code

> **Today's big idea:** You can **command an AI with real code**! The AI model lives behind
> an **API**, and a few lines of **Python** can ask it for a reply. Today your assistant
> comes **alive** — same code, *your* personality.

### 4 words to know

| Word | What it means |
|------|---------------|
| **Python** | A friendly coding language that reads almost like English |
| **Colab** | A website to run Python in your browser — nothing to install |
| **Cell** | One block in the notebook you can run ▶️ |
| **API** | The way our code **asks the AI model** for a reply (the model lives behind it) |

> 🔑 **About the key:** code needs a secret **API key** to use the AI — like a password.
> **The facilitator holds it. You never see or type it.**
