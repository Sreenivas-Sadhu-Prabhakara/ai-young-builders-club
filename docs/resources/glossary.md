# Kid-friendly glossary

*Every AI word we use in the club, explained the way we'd say it to an 11-year-old.*
Hand this out, or pull it up when a word comes up. Words are grouped by when you'll first
meet them.

## The big ideas (Week 1)

**Artificial Intelligence (AI)**
: Computer programs that can do things that usually need human smarts — like recognizing a
  cat in a photo or writing a sentence. The trick: they **learn from examples** instead of
  being given step-by-step rules.

**Machine Learning**
: The main way we make AI today. Instead of a human writing every rule, we show the
  computer **lots of examples** and it figures out the patterns itself.

**Training**
: Showing an AI tons of examples so it can learn the pattern. Like practicing free throws
  until you get good — the AI "practices" on examples.

**Model**
: The thing you get *after* training — the trained "brain" that makes predictions. When you
  trained Teachable Machine, the result was your **model**.

**Training data**
: The pile of examples used to train a model. If the examples are good and varied, the model
  is good. If they're biased or limited, the model is too. *Garbage in, garbage out.*

**Pattern**
: A thing that repeats. AI is basically a super-powered pattern spotter — that's its whole
  superpower (and its weakness).

**Label**
: The "answer" attached to an example. If you train it on cat photos labeled "cat," the
  label is "cat." (In Teachable Machine, your classes — like "thumbs up" — are labels.)

**Classify / Classifier**
: Sorting things into groups. A classifier looks at something new and says which group it
  belongs to ("that's a thumbs up").

## Talking machines (Week 2)

**Chatbot**
: An AI you talk to in words. You type something, it types back.

**Large Language Model (LLM)**
: The kind of AI behind chatbots. It learned from a huge amount of text and is really good
  at **predicting the next word** — which is how it writes whole answers.

**Predicting the next word**
: How a chatbot actually works. It doesn't "know" the answer — it guesses the next word,
  then the next, then the next, super fast, based on patterns it learned.

**Prompt**
: What you type to an AI to tell it what you want. A clear prompt → a better answer.

**System prompt**
: A special hidden instruction that sets an AI's job and personality before you even start
  chatting — like its "character sheet." You'll write one for your assistant.

**Token**
: A chunk of text the AI works in — usually a word or part of a word. AI reads and writes in
  tokens, not whole sentences.

**Hallucination**
: When an AI makes something up and says it **confidently**, even though it's wrong. It's not
  lying on purpose — it's just predicting words that *sound* right. Always check important
  stuff.

**Bias**
: When an AI is unfair or lopsided because its training examples were lopsided. Example: if
  it mostly saw pictures of one kind of "doctor," it might assume all doctors look that way.

**Generative AI**
: AI that **makes new things** — text, images, music — instead of just sorting or labeling.

**Image generator**
: A generative AI that makes pictures from a text prompt.

## Real code (Week 3)

**Code**
: Instructions written for a computer in a language it understands. We use a tiny bit of
  **Python**.

**Python**
: A popular, friendly programming language. It reads almost like English, which is why
  beginners (and AI engineers) love it.

**Colab (Google Colab)**
: A website where you can write and run Python in your browser — no installing anything. We
  use it to bring your assistant to life.

**Notebook**
: A document in Colab that mixes notes and runnable code in little blocks called **cells**.

**Cell**
: One block in a notebook. You can run a cell to make its code happen. ▶️

**API**
: A way for one program to ask another program to do something. We use an API to let our few
  lines of Python **ask a big AI model** for a reply.

**API key**
: A secret password that lets your code use an AI service. Like a key to a building —
  **never share it or paste it where others can see**. (In the club, the facilitator handles
  the key — you never touch it.)

**Variable**
: A labeled box that holds a value in code, like `name = "Robo"`. Change the box, change what
  the program does.

## Smart & safe (Week 4)

**Deepfake**
: A fake photo, video, or voice made by AI that looks/sounds real. Powerful for fun — and
  dangerous when used to trick people.

**Guardrail**
: A rule that keeps an AI from doing something it shouldn't (like refusing rude requests).
  You'll add one to your assistant.

**Privacy**
: Keeping your personal information to yourself. With AI, that means **never** typing secrets
  into a tool (club rule #1).

**Responsible AI / AI ethics**
: Thinking about whether an AI is fair, safe, and used kindly — not just whether it's cool or
  clever.

**Misinformation**
: False information that spreads — sometimes made or boosted by AI. Being able to check
  sources is your superpower against it.
