# FAQ & troubleshooting

*Answers to the questions facilitators and kids actually ask.*

## For facilitators

### Do I need to know how to code or how AI works?
No. The run-sheets give you talking points, the kid questions you'll get (with answers), and
fallbacks. The honest answer "I don't know — let's find out together" models exactly the
curiosity we want. Doing each activity yourself once beforehand is the only prep you need.

### What if a kid finishes way ahead of everyone?
Every activity has a **"Go Further"** challenge. Point them there, or make them a co-helper
for a kid who's stuck (great for them, great for you).

### What if a kid is totally stuck or frustrated?
Pair them with someone, or sit with them for the capstone step (that's the part that
matters most). Progress on *their* assistant beats finishing the side activity.

### The group is mixed ages / abilities. How do I pace it?
Run the **core path** for everyone; let fast kids drift into "Go Further." The run-sheets
have buffer time. It's fine to not finish a side activity — protect the capstone step and
the wrap.

### How strict is the one-hour limit?
Treat it as real. The run-sheets are timed so that even if the hook or hands-on runs long,
the **capstone step and wrap always happen** (cut the side activity short, not the capstone).

### What's the single most important thing each week?
The **capstone step** (0:50–0:58). If everything else falls apart, get each kid one step
further on their own assistant and the week succeeded.

## When tech breaks

A calm fallback beats a scramble. Each facilitator guide has an unplugged version; here are
the general moves:

| Problem | Quick fix | Fallback |
|---------|-----------|----------|
| Wifi down / slow | Switch to facilitator screen only; kids direct, you drive | Run the **unplugged** activity from the guide |
| A tool is down / over quota | Use your pre-saved screenshots/examples | Switch to the paper version of the activity |
| Webcam won't work (Week 1) | Use the image-upload mode instead of live camera | Use sample images you saved earlier |
| Colab won't run a kid's notebook (Week 3) | Have them use **your** working copy on the projector | Use the notebook's **canned-response mode** |
| API key/model error (Week 3) | Switch the notebook to canned-response mode | Demo your working copy live |
| A kid sees inappropriate output | Stay calm, close it, reassure them (rule #3) | Switch that kid to the unplugged version; note it |

!!! tip "Always have a canary and a screenshot stash"
    Before each session, confirm the day's tool works on one laptop, and keep a folder of
    screenshots of successful results. That folder turns any outage into a smooth demo.

## Questions kids ask (and good answers)

**"Is the AI alive / does it think?"**
No. It's a very good pattern-matcher. It learned from a huge pile of examples and predicts
what usually comes next. It doesn't understand or feel anything — even when it sounds like
it does.

**"Does the AI know it's wrong when it's wrong?"**
No. It can sound totally confident and still be completely wrong. That's called a
*hallucination* — and it's exactly why we check important things.

**"Where does it get its answers?"**
From patterns in the huge amount of text/images it was trained on — not from looking things
up like a search engine (unless it's specifically connected to one). So it can be out of
date or just make things up.

**"Can it learn who I am?"**
Don't give it the chance — that's club rule #1. Many tools keep what you type. So we never
put real personal info in.

**"Will AI take everyone's jobs / is it dangerous?"**
It changes jobs more than it deletes them, and like any powerful tool it can be used well or
badly. That's why learning to use it wisely — what you're doing right now — matters.

**"Why did it draw the hands/text weird?"**
Image AIs predict what pixels usually look like; they don't *understand* that hands have
five fingers. They're guessing from patterns, so they get fiddly details wrong.

**"Can I use this at home?"**
Talk to your grown-up about which tools are okay, and bring the same club rules with you —
especially "no secrets" and "AI can be wrong."
