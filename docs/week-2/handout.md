## 💬 Talking machines

> **Today's big idea:** A chatbot works by **predicting the next word** — then the next, then
> the next, super fast. Your **prompt** is the steering wheel. And it can be **confidently
> wrong**!

### 6 words to know

| Word | What it means |
|------|---------------|
| **Chatbot** | An AI you talk to in words |
| **LLM** | The giant "next-word predictor" behind it |
| **Prompt** | What *you* type to steer it |
| **System prompt** | The hidden "character sheet" you give it |
| **Hallucination** | When it makes something up — confidently! |
| **Bias** | When it's unfair because its examples were lopsided |

*(More in the [glossary](../resources/glossary.md). Keep the
[prompting cheat sheet](../resources/prompt-cheat-sheet.md) next to you!)*

## 🦾 Activity: prompt it, catch it, draw with it

> 🔒 **Rule #1:** No secrets in prompts! No real full name, address, school, or passwords.

**(a) Weak prompt → strong prompt** — use **R-T-F-C** (Role · Task · Format · Check):

- [ ] Type a weak prompt like *"tell me about dogs"* — read the boring answer
- [ ] Rebuild it: add a **Role**, a clear **Task**, a **Format**, and a **Check**
- [ ] Compare! Strong prompt wins 🚀

**(b) Catch a hallucination:**

- [ ] Confidently ask about something fake, e.g. *"the Great Sock War of 1850"*
- [ ] Watch it **invent facts** with total confidence 😵
- [ ] Now add a check: *"Only answer if this is real."* Does it behave better?

**(c) Make images & spot the flaws:**

- [ ] Try the **same** image prompt in **3 styles** (cartoon, pixel art, watercolor)
- [ ] Look closely at the **hands** and any **text** — find the wonky bits! 🖐️

**Write what you notice:**

The strong prompt was better because... _____________________________________________

When I asked about something fake, the chatbot... ___________________________________

The image AI was really bad at drawing... __________________________________________

> 💡 **Big lesson:** A chatbot **predicts likely words** — *likely* isn't the same as *true*.
> It can be **confidently wrong** (hallucination) and **unfair** (bias). *Garbage in, garbage
> out!*

**🚀 Go Further:** Politely try to confuse the bot · compare two AIs on the same prompt · run
one image prompt in 3 styles.
